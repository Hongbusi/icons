# Icons

Forked from [element-plus-icons](https://github.com/element-plus/element-plus-icons).

## Preview

[Icons](https://icons.hongbusi.com)

## Install

``` bash
pnpm add @hongbusi/icons-vue
```

## Usage

``` vue
<script setup lang="ts">
import { Github } from '@hongbusi/icons-vue'
</script>

<template>
  <Github />
</template>
```
